Primeira pedra do Lume. 🌱

Esta versão estabelece a base do seu segundo cérebro local:

- **Formato OKF** — suas notas são arquivos markdown legíveis, com tipo, tags e links. Nada de banco proprietário, nada preso na nuvem.
- **App nativo** em Rust + Tauri: leve, rápido e 100% offline.
- **Leitor do cérebro** — uma primeira tela que lê e lista suas notas.
- **Instaladores automáticos** para Windows, macOS e Linux, gerados a cada versão.

Ainda é só o começo: a IA que lê e edita o cérebro com você vem nas próximas versões.
